# Flight-Stick

### This project's purpose was to use a joystick module paired with buttons and an Arduino Pro Micro to serve as a makeshift flight stick for the X-Plane 12 flight simulator.

### The whole point of this project was to learn how to make makeshift joysticks for games such as flight simulators and maybe even racing games. This project also improved my CADDING and programming skills. I learned how to research different parts and find the best one for my use case, and how to improvise with what I have.
